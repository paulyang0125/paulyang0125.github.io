I contributed to build enforcing and corrective feedback loop from Arm ecosystem communities to co-develop the specifications and drive the engineering team to implement the test suite aligning with the specs.
The BSA/BBR Intro
The Arm Base System Architecture (BSA) and Base Boot Requirements (BBR)specification defines a hardware and FW system architecture, based on Arm 64-bit architecture, that system software can rely on. System software includes operating systems, hypervisors, and firmware. The requirements and runtime features required by the BSA ensure that users can install, boot, and run generic operating systems and hypervisors.

Intel Virtualization Technologies - VT-x, VT-d

VT-x is the generic virtualization of the CPU. EPT is a feature that is usually enabled together with VT-x that allows proper virtualization.

My contribution was to enable x86 BIOS vendors' implementations such as AMI Aptio to use those CPU virtualization ISA and developed the validation kit and user guide for x86 PC OEMs/ODMs.
VT-x is the generic virtualization of the CPU. EPT is a feature that is usually enabled together with VT-x that allows proper virtualization (VT-x without EPT cannot virtualize 64-bit OS properly). VT-d is used to implement something called PCIe Passthrough. That is, the virtual machine has direct access to a host PCIe device. Typically, it is a GPU and a NIC that is passed through for performance issues.
